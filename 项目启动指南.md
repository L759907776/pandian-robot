# 项目启动指南

## 📋 项目概述

这是一个**纯静态网站项目**，基于HTML、CSS、JavaScript开发，主要用于展示盘点机器人产品。

## 🚀 快速启动（选择一种方式即可）

### 方法一：Python内置服务器（推荐⭐⭐⭐⭐⭐）

```bash
# 1. 进入项目目录
cd d:\AiCodeTest\pandian2

# 2. 启动Python HTTP服务器
python -m http.server 8000
```

**访问地址**：http://localhost:8000

---

### 方法二：Node.js serve（推荐⭐⭐⭐⭐）

```bash
# 1. 全局安装serve
npm install -g serve

# 2. 进入项目目录并启动
cd d:\AiCodeTest\pandian2
serve -s . -p 8000
```

**访问地址**：http://localhost:8000

---

### 方法三：PHP内置服务器

```bash
# 1. 进入项目目录
cd d:\AiCodeTest\pandian2

# 2. 启动PHP服务器
php -S localhost:8000
```

**访问地址**：http://localhost:8000

---

### 方法四：VS Code Live Server插件（推荐⭐⭐⭐⭐）

1. 在VS Code中安装"Live Server"插件
2. 右键点击 `index.html` 文件
3. 选择 "Open with Live Server"

---

## 📝 开发说明

### 文件编辑
- **HTML**: 编辑 `index.html` 修改页面内容
- **CSS**: 编辑 `css/style.css` 修改样式
- **JavaScript**: 编辑 `js/script.js` 修改交互功能
- **图片**: 替换 `images/` 文件夹中的图片

### 热重载
使用Live Server或其他支持热重载的工具时，修改文件后浏览器会自动刷新。

## 🔧 故障排除

### 问题1：端口被占用
**解决方案**：更换端口号
```bash
python -m http.server 8080  # 使用8080端口
```

### 问题2：图片不显示
**检查**：
1. 确保图片文件在 `images/` 文件夹中
2. 检查图片路径是否正确
3. 使用开发者工具(F12)查看控制台错误

### 问题3：样式丢失
**检查**：
1. 确保 `css/style.css` 文件存在
2. 检查HTML中的CSS引用路径
3. 清除浏览器缓存后重新加载

## 📞 技术支持

如遇到启动问题，请检查：
1. Python/Node.js/PHP是否正确安装
2. 防火墙是否阻止了端口访问
3. 项目文件是否完整（特别是index.html）
